To open the calculator, install node.js software, and restart your computer after installation. After restarting the computer, press the window icon on the keyboard, and press the R button at the same time to open the "Run" app
Run button, a black window will pop up, please enter: cd /d [calculator file extraction path] in it. After that, type: npm init -y. Then type: npm install express and open the run.bat
It's time to open it.
